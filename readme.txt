== Installation ==

The “WP Force AMP” plugin requires “AMP” plugin for WordPress by “Automattic”.  So if you haven’t installed AMP plugin yet, kindly follow these steps:

1. Log in to your WordPress admin and navigate to Plugins > Add new.
2. Inside “Search Plugins” field, type “AMP”. From the results, install the AMP plugin as found in this URL: https://wordpress.org/plugins/amp/
3. After installation, activate the plugin via Plugins > Installed Plugins > AMP

Once that is done, to install the “WP Force AMP” plugin, follow these steps:

1. Log in to your WordPress admin and navigate to Plugins > Add new.
2. Inside “Search Plugins” field, type “Adsense for AMP”.
3. After installation, activate the plugin via Plugins > Installed Plugins > WP Force AMP
Once the plugin is activated, you can navigate to the settings page for “WP Force AMP”.


== Frequently Asked Questions ==

= How to test the changes? =
To test the changes, save the settings and view any blog post on a mobile device, eg: <i>https://yourdomain.com/your-sample-article/</i>. 

'WP Force AMP' plugin would automatically redirect you to AMP version of the blog post, eg: <i>https://yourdomain.com/your-sample-article/amp/</i>.

Note: Don't see any changes? try to clear the site/browser cache and refresh.
